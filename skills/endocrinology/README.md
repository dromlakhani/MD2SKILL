# Endocrinology Skills

Clinical decision support tools for endocrine conditions, with a focus on diabetes management.

---

## Topics

### 🦶 Diabetic Foot
Skills for assessment and management of diabetes-related foot infections (DFI), based on the IWGDF/IDSA 2023 guidelines.

| Skill | What it does |
|---|---|
| [DFI Severity Classifier](diabetic-foot/dfi-severity-classifier/) | Classify infection grade (1–4) + osteomyelitis screen |
| [DFI Empiric Antibiotic Selector](diabetic-foot/dfi-antibiotic-selector/) | Select antibiotic by grade + MRSA risk |

> Use these two skills together — Classifier first, then Antibiotic Selector.
